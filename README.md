## CPF Validator API

API simples para validação de CPF, desenvolvida em Node.js com Express, seguindo uma arquitetura organizada e separação de responsabilidades.
O projeto tem como objetivo servir como base de estudo e portfólio, demonstrando boas práticas na construção de APIs REST, como rotas bem definidas, lógica isolada e código de fácil manutenção.

---

##  Tecnologias Utilizadas

- Node.js
- Express
- JavaScript
- Git & GitHub
